Train Tracked

A 'better' UK train tracking app. Created as part of a Level 5 Computer Science project at the University of Portsmouth

Developed using Flutter, Dart and the Rail Delivery Group's delightful Darwin API

Now Open Source! The moderation period has ended and marks have been released, so I can now safely release this project as open source software!
The front end and back end are both available now on Github!

I intend to release Train Tracked on the Google Play Store SoonTM, and as a direct download now (Get it here, updated 17-07-2024). However, an iOS release is looking unlikely as I do not own any Apple devices (Why Apple, do I need a mac to publish for iOS?)
