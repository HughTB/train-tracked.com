Train Tracked

A 'better' UK train tracking app. Created as part of a Level 5 Computer Science project at the University of Portsmouth. Developed using Flutter, Dart and the Rail Delivery Group's delightful Darwin API

Now open source!

The moderation period has ended and marks have been released, so I can now safely release this project as open source software!
The front end and back end are both available now on Github!
